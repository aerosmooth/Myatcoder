# MyAtCoder

## 概要

AtCoderのコンテスト参加時の提出コードや、競技プログラミングの学習記録を管理するためのリポジトリです。
実装は基本的にすべて Python で行っています。

## ディレクトリ構成

- **`Rated_Contest/`**
  - 参加したレーティング対象コンテスト（日付別）のコードを格納しています。
  - コンテストによっては、本番での自分の提出コード（`mysubmission/`）と、コンテスト後の復習コード（`answer/`）を分けて管理しています。

- **`chap1/` 〜 `chap9/`**
  - 「[競技プログラミングの鉄則](https://github.com/E869120/kyopro-tessoku)」に基づく、各章・各テーマごとの演習問題の答えコード等を格納しています。
- **その他のファイル**
  - `change_user.sh`: Gitの過去のコミット履歴のユーザー情報を修正するためのユーティリティスクリプトです。
  - `.gitignore`: 特定のディレクトリ（`kyopro-tessoku`など）の管理を除外する設定です。

## 使用言語

- Python 3
